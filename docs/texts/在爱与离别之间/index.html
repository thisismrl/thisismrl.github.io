在爱与离别之间

Poem / 2026-06-04

风吹落你的草编帽，
你下意识伸出手，
在你惊呼的刹那，
黄昏透过我们身体。
此刻你是否会想到，一切终将离去？

你躺在枕头另一面，
静默的游丝在我们肺间缠绕。
我闭上眼睛，望向你，
在你屏住呼吸之前，
此刻你是否会想到，一切终将离去？

你在结尾的祝福里删去一个词，
然后写完了这封信。
窗外的树影掠过信纸，
纸上墨迹尚未干涸。
此刻你是否会想到，一切终将离去？

特罗姆瑟的夜色降临时，
游轮正从不冻港驶向大西洋。
暖流蒸腾起云雾，
时间沿着海浪流向天空。
亲爱的，此刻你仍未离去。

龍梓洋 Charles Long 2026
